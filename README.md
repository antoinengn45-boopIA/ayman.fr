## 🚀 Modules Système

Le terminal est divisé en plusieurs zones de contrôle critiques :

*   **01_SANDTECH :** Accès direct au moteur de développement de jeux et au studio SandBlock.
*   **02_ARCHIVE :** Dépôt central des frameworks et outils Open Source hébergés sur GitHub.
*   **03_COMMS :** Protocole de communication chiffré via Tally pour toute requête officielle.

---

## 🛠️ Stack Technique

Le projet est conçu pour une efficacité maximale et une latence minimale :

*   **Frontend :** `HTML5`, `Tailwind CSS` (pour le stylisme atomique).
*   **Logic :** `JavaScript` (Gestion de la console système et animations).
*   **Analytics :** `Umami` (Suivi de télémétrie léger et respectueux).

---

## ⚙️ Installation & Déploiement

Ce projet est une application web statique. Pour le déployer sur ton serveur personnel :

1.  **Cloner le dépôt :**
```bash
    git clone [https://github.com/ton-repo/ayman.fr](https://github.com/ton-repo/ayman.fr)
    ```
2.  **Configuration :**
    *   Aucun processus de build lourd n'est requis.
    *   Assure-toi que les clés d'accès `data-tally-open` et `data-website-id` sont correctement configurées dans le fichier `index.html`.
3.  **Hébergement :**
    *   Déploie sur **Vercel**, **Netlify** ou **GitHub Pages** pour une mise en ligne immédiate.
