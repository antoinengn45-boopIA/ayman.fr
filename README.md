# Aywen Studio

Plateforme centralisée présentant mes projets technologiques, mes développements de jeux et outils d'automatisation.

## 🚀 À propos
**Aywen** est un hub personnel regroupant :
- **Framework Web :** Outils open-source pour le développement front-end.
- **SandBlock Studio :** Conception et développement de jeux vidéo.
- **Bot Management :** Solutions d'automatisation pour Discord.

## 🛠 Stack technique
- **Frontend :** HTML5, Tailwind CSS
- **Framework :** Node.js / Express
- **Icônes :** Font Awesome
- **Intégration :** Tally.so (Formulaires)

## 📦 Installation
1. Clone le projet : `git clone https://github.com/ton-user/aywen.git`
2. Installe les dépendances : `npm install`
3. Lance le serveur : `node server.js`
